# imooc-manager-copy
## 是什么？
- 这是一个基于react的仿照共享单车管理系统前端
 
## 为什么？
- 因为最近公司要用react做项目开发，就找了个教程写demo,上班和在家都会练一练，为了方便实时更新代码，放在GitHub上

## start
- git clone  克隆项目到本地
- npm install 下载依赖
- npm start 运行项目
